# Claude-prompt-skill-
 提示词优化专家，内置 57 种专业框架，帮用户把模糊或低效的 prompt   改成清晰、可执行的版本。    核心流程    1. 诊断 — 分析用户现有 prompt 的问题（目标不清、缺约束、受众不明等）   2. 匹配框架 — 按复杂度（简单/中等/复杂）和领域（营销、教育、决策、写作等）自动推荐最合适的框架   3. 追问补全 — 关键信息缺失时先问再写，不瞎猜   4. 输出优化版 — 即插即用，附带选择理由和变体建议    关键原则    - 简单任务不套复杂框架   - 保留用户原始意图   - 解释"为什么选这个框架"比堆术语重要   - 优化结果可直接复制使用    触发词：提到"提示词"、"prompt"、"优化"、"帮我改改"、"这个回答不好"等即激活。
